name: 纯文本格式适用范围
description: 纯文本格式规则只适用于项目产出文档，不适用于规则系统本身
metadata:
  type: feedback

规则：所有 .md 文件使用纯文本格式，不写 Markdown 语法。

适用范围：
- 适用：新项目初始化时生成的记忆文件、文档、任务卡等所有产出文件
- 不适用：本规则系统本身（项目开发步骤.md、CLAUDE.md），维持 Markdown 格式以便编辑

Why: 用户选择此范围，认为规则系统需要 Markdown 格式便于阅读和编辑，而项目文档用纯文本更简洁。
How to apply: 初始化新项目时，生成的所有 .md 文件使用纯文本格式。更新规则文档时继续使用 Markdown。
