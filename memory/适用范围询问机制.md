metadata:
  type: project

本规则系统新增「适用范围」条款，明确规则仅适用于 IT 系统项目（网站、App、小程序、PC 系统等）。

**改动内容：**
- 在 `项目开发步骤.md` 开头新增「适用范围」章节，列举适配和不适配的项目类型
- 在初始化流程中新增「1.0 适用范围确认」步骤，每次初始化前先问用户是否启用
- 同步更新 `CLAUDE.md` 中的初始化流程，增加适用范围确认作为第一步
- 版本从 v1.7 更新至 v1.8

**Why:** 原规则未声明适用边界，进入非 IT 类项目时也会自动执行全套初始化，产生不必要的工作。

**How to apply:** 每次收到"开始新项目"或"用规则初始化项目"时，先询问用户当前项目是否属于 IT 系统项目，再由用户判断是否继续。
